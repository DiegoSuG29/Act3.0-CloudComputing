# Act3.0 - Cloud Computing
---
## Repositorio para la actividad 3.0 de Cloud Computing

Proyecto en colaboración con Fuego Azteca con el objetivo de utilizar los datos de las plataformas de venta de la compañía (Amazon, Mercado Libre) para generar un proceso que tome esa información, la analice y utilizando PowerBI para tomar decisiones.
- Dashboard interactivo
- Gráficos
- Despliegue de información en el ERP que ya utilizan
- Rediseño de la página web
